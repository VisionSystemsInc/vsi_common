VSI Common is a set of tools and scripts that are not specific to any project, but useful to have on any project. The most significant of these tools being J.U.S.T..

VSI_COMMON_DIRÔÉÅ

The location of the root of the VSI common code repository. Most scripts will determine VSI_COMMON_DIR automatically, and use it to find other scripts in VSI Common. VSI_COMMON_DIR is almost always overrideable, should you need to reference a different location/version of VSI Common.
