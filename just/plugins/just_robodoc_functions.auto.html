J.U.S.T. Robodoc Functions

Warning

This module is deprecated in favor of J.U.S.T. Sphinx Docs Functions. Robodocs will not be updated and maybe removed in the future.

just_robodoc_functions.bsh
Arguments:

  • [$1] - Optional docker location of robodoc rc files, wrt /src dir. Default: /src/robodoc.rc

  • SRC_DIR - Source directory. Default: $VSI_COMMON_DIR

  • DOC_DIR - Document directory. Default: $VSI_COMMON_DIR/docs

robodoc - target that will make robodoc documentation
